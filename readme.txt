== Description ==

Libre.fm Now Playing fetches the currently playing track from your [Libre.fm](http://libre.fm "Libre.fm") account and displays it in a widget, optionally linking it to Libre.fm.

== Installation ==

1. Download the plugin archive.
2. Uncompress the archive.
3. Upload archive contents to '/wp-content/plugins/'.
4. Activate the plugin through the 'Plugins' menu in WordPress.
5. Add the widget to your theme through the 'Widgets' menu in WordPress.
6. Enter your Libre.fm username in the widget. Save.

== Frequently Asked Questions ==

= What is Libre.fm? =

Libre.fm is a music community website similar to last.fm, but based on ideals of free software and promoting the use of free licenses in music.
